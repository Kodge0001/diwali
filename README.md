Celebrate the Festival of Lights with a custom-generated greeting! This interactive web app allows users to create and share beautiful Diwali cards with personalized messages.

## ✨ Features
- **Custom Themes**: Choose from multiple festive background themes.
- **Personalized Messages**: Add your name and a custom wish for your loved ones.
- **Interactive Animations**: Festive animations including fireworks and glowing diyas.
- **Shareable Links**: Easily share your custom greeting with friends and family.

## 🚀 Live Demo
Create your greeting here: **[kodge0001.github.io/diwali](https://kodge0001.github.io/diwali/)**

## 🛠️ Build Details
- **Frontend**: Vanilla JavaScript for dynamic DOM manipulation.
- **Styling**: CSS3 animations and gradients for a festive look.
- **Deployment**: Automated via GitHub Actions.
